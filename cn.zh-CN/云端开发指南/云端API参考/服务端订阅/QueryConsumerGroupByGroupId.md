# QueryConsumerGroupByGroupId

使用AMQP服务端订阅时，可以调用该接口根据消费组ID查询消费组详情。

## 限制说明

单阿里云账号调用该接口的每秒请求数（QPS）最大限制为5。

**说明：** RAM共享阿里云账号配额。
